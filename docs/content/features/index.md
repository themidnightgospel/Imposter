# Features

Imposter covers methods, properties, indexers, and events with a consistent fluent API.

- Methods: returns, callbacks, sequencing, throws, verification, async, ref/out/in.
- Properties: getter/setter setups, sequencing, base forwarding, verification.
- Indexers: getter/setter setups with `Arg<T>` matchers, base forwarding.
- Events: subscription verification, raising, interceptors, invocation counts.
